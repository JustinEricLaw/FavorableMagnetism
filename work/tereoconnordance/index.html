Tere O'Connor Dance Website

Web Development, Responsive Design

Responsive website for Tere O'Connor Dance, a modern dance company based in New York City. I contributed all design, development, and content strategy for this site, as well as a custom Wordpress theme. Using a mobile-first approach, this site was built to provide an intuitive user experience and fast loading times for all devices. I also customized the Wordpress admin area to make site updates much easier for the company.
