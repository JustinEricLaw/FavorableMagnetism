Tere O’Connor Dance Website

Web Development, Responsive Design

Responsive website for Tere O’Connor Dance, a modern dance company based in New York City. I was the sole designer and developer for this project. I built a custom Wordpress theme and established content strategy. Using a mobile-first approach, this site was built to provide an intuitive user experience and fast loading times for all devices. I also customized the Wordpress admin area to make site updates easier for the company.
