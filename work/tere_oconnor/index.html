Tere O’Connor Dance

Interface Design | Web Development

Responsive website for Tere O’Connor Dance, a modern dance company based in New York City. I was the sole designer and developer for this project, building a custom Wordpress theme and establishing content strategy. The site highlights upcoming performances, media, information about the company, as well as various teaching resources. I also customized the Wordpress admin area to make site updates more user-friendly for the company.
